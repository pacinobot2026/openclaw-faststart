## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Add environment variable:
   - `STRIPE_SECRET_KEY` = (your Stripe key)
4. Deploy
